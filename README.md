# Anki Deck Generator

Anki Deck Generator is a full-stack web application that allows users to create and organize Anki flashcards from text input. Designed for students, educators, and lifelong learners, it streamlines study workflows and helps manage decks efficiently.

---

## 🚀 Features

- Generate Anki flashcards from text
- Organize flashcards into decks and categories
- Full-stack architecture for scalability
- Future-ready for exporting decks and AI-based card generation
